# =============================================================================
# SRV Quantamental Decision Support System — Python dependencies
#
# System overview:
#   Sector-rotation / statistical-arbitrage DSS built on SPDR sector ETFs
#   (XLC–XLU) vs SPY.  Data is ingested from Financial Modeling Prep (FMP)
#   and stored as Parquet artifacts in data_lake/.  The analytics core
#   (analytics/stat_arb.py) runs rolling out-of-sample PCA, EWMA vol,
#   macro betas, and correlation-structure diagnostics.  Attribution scores
#   (SDS / FJS / MSS / STF / MC) are computed per sector and surfaced through
#   a Dash / Plotly dashboard with a Hebrew RTL UI.  A data-ops layer
#   (data_ops/) provides freshness, coverage, and validation health checks
#   that are wired into the dashboard's Data Health tab.
#
# Python requirement: >= 3.11
# Versions pinned to stable, production-grade releases as of early 2026.
# No ML frameworks (torch / transformers / jax) are required.
# =============================================================================
